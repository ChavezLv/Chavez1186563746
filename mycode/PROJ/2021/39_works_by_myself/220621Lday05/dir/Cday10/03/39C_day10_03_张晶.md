- (a)

```c
void doufind(int*arr, int a,int left,int right) {	
	if (a<arr[0] || a>arr[N-1]) {
		printf("The num is not find!\n");
		return;
	}
	while (left <= right) {
		int mid = (left + right) / 2;
		if (a > arr[mid]) {
			left = mid + 1;
		}
		else if (a < arr[mid]) {
			right = mid - 1;
		}
		else {
			printf("%d能找到，下标为：%d\n", a, mid);
			return;
		}		
	}
}

```

- (c)

```c
哈希冲突：不同的数据返回同一个哈希值
解决冲突：
    开放寻址法：根据根据碰撞次数重算哈希值
    优点：不用分配内存
    缺点：鸠占鹊巢
    链表法：将数据的信息作为链表结点插入到链表中
    缺点：要分配内存
    优点：方便管理相同哈希值的数据
```
