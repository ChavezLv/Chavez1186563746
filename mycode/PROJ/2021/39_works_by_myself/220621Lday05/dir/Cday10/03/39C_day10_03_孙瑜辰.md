## 答案

- (a)

```c
int BinarySearch(int *arr, int x, int length) {
		int left, right, mid;
		left = 0;
		right = length - 1;
		while (left <= right) {
			mid = (left + right) / 2;
			if (x < arr[mid]) {
				right = mid - 1;
			}
			else if (x > arr[mid]) {
				left = mid + 1;
			}
			else {
				return mid;
			}
		}
		return -1;
}

```
