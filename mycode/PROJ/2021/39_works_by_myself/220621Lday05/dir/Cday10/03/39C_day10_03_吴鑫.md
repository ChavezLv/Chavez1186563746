## 答案

- (a)

```c
int binarySearch(int* arr, int low, int high, int targetNum) {
	int mid;
	while (low <= high) {
		mid = (low + high) / 2;
		if (arr[mid] > targetNum) {
			high = mid - 1;
		}
		else if (arr[mid] < targetNum) {
			low = mid + 1;
		}
		else {
			return mid;
		}
	}
	return -1;
}

```
